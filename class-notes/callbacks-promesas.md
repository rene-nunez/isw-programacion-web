JavaScript consta de diversas formas para resolver el problema de asincronía.

## CallBacks

Es una de las técnicas y forma más común para el control de la asincronía dentro de JavaScript.

Contiene:

- Función A de la que uno de sus argumentos es una función B que por convención recibe el nombre de callback.
- Callback recibe 2 argumentos, el primero es una referencia a un error, el segundo argumento contiene el resultado de la función A que está recibiendo el callback.
